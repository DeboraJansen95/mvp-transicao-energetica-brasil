# Transição Energética no Brasil
## MVP — Pós‑Graduação em Ciência de Dados e Analytics | PUC‑Rio
### Autora: Debora Jansen

Este projeto analisa a evolução do consumo energético brasileiro e desenvolve modelos para projetar tendências futuras, utilizando técnicas de análise de dados e séries temporais. A pesquisa utiliza o dataset do *Our World in Data* para compreender a dinâmica da matriz energética nacional e o avanço das fontes renováveis no país.

---

## Objetivo
Investigar a evolução do consumo energético brasileiro e projetar tendências futuras utilizando técnicas de análise de dados e séries temporais, contribuindo para discussões sobre sustentabilidade, planejamento energético e políticas públicas.

---

## Metodologia

### 1. Coleta e Carregamento dos Dados
- Utilização do dataset oficial do *Our World in Data*  
- Importação via GitHub para garantir reprodutibilidade

### 2. Preparação e Limpeza
- Filtragem dos dados referentes ao Brasil  
- Tratamento de valores ausentes  
- Organização da série temporal anual

### 3. Análise Exploratória
- Visualização de tendências históricas  
- Comparação entre fontes renováveis e fósseis  
- Identificação de padrões e mudanças estruturais

### 4. Modelagem e Previsão
- Aplicação de técnicas de séries temporais  
- Construção de modelos preditivos  
- Projeção dos próximos anos

---

## Dados Utilizados
- Fonte: *Our World in Data — owid-energy-data.csv*  
- Variáveis analisadas:  
  - Consumo de energia renovável  
  - Participação de renováveis  
  - Consumo de combustíveis fósseis  
  - Emissões de CO₂  
  - Energia per capita  
  - Geração elétrica total
